# 7 家基于 Python 的公司我们很高兴加入 FT1000

> 原文：<https://www.stxnext.com/blog/7-python-powered-companies-ft1000/>

 STX Next 连续第二年在《金融时报》1000 强的  [中被评为欧洲增长最快的公司之一。](https://www.ft.com/content/238174d2-3139-11e9-8744-e7016697f225)

但是我们只是众多选择 Python 作为他们选择的编程语言来建立快速扩展业务的人中的一员。

以下是 2019 年 FT1000 中我们最喜欢的几家基于 Python 的公司。 

#### 1.交货

[https://deliveroo.co.uk/](https://deliveroo.co.uk/)

![deliveroo.png__730x250_q85_crop_subsampling-2_upscale](img/c6869ce0734b635e7700c633a9ce45c5.png)

*   **排名第二**
*   **绝对收入增长:15，749%**
*   **国家:英国**
*   **部门:食品&饮料**

在今年的 FT1000 排名中，无需寻找第一家使用 Python 的公司。Deliveroo 今年以惊人的增长速度排名第二:15，749%！

更令人印象深刻的是员工数量的增长。2014 年至 2017 年间，Deliveroo 从只有 28 人增长到 1664 人。

请记住，该公司成立于 2012 年，尽管在英国市场上面临其他竞争者的竞争，但增长如此之快。

Deliveroo 的使命是让你以不同的方式思考送餐对你意味着什么。用他们自己的话说:“这不再是鸡肉炒面和沙发上的夜晚，这是你最喜欢的当地餐厅，这是一个晚宴，一次约会。”

Deliveroo 旨在通过 Deliveroo Editions 等服务来完成这一使命，deliver oo Editions 是一种定制厨房，提供精选餐厅。

今天，你可以通过 Deliveroo 在 13 个国家的 200 个城市订购美味的食物。我们焦急地等待他们在波兰出现。

##### **Deliveroo + Python**

Deliveroo 自己承认他们的  “栈是基于 Python 的。”  例如，希望申请 deliver oo[数据科学家职位](https://www.glassdoor.com/Interview/Deliveroo-Data-Scientist-Interview-Questions-EI_IE1053365.0,9_KO10,24.htm) 的候选人会很快发现 Python 经验会给他们带来优势。这并不奇怪，因为 Python 以其  [丰富的科学库](/stx-new-blog)而闻名。

此外，Deliveroo 运行一个 GitHub 存储库，其中有 1 个不起眼的 Python 条目:[【rds-snapshot-tool】](https://github.com/deliveroo/rds-snapshot-tool)。

#### 2.自立门户

[https://housekeep.com/](https://housekeep.com/)

![housekeep.png__730x250_q85_crop_subsampling-2_upscale](img/c6edf84d1bb573fadc476ee0b1fc6d34.png)

*   **排名第六**
*   **绝对收入增长:6100%**
*   **国家:英国**
*   **部门:科技**

作为我们名单上最年轻的公司，Housekeep 在很短的时间内取得了惊人的增长。

Housekeep 成立于 2014 年，为成千上万的伦敦人提供优质的家居清洁服务，并专注于行业最佳的客户服务。

使用 Housekeep，你可以预约值得信赖的清洁工上门服务，他们会保持你的公寓整洁，并提供额外的服务，如清洁冰箱和烤箱。为了额外的方便，你甚至可以预订你的清洁工来打扫你的家，而你在工作。

最棒的是:从预订到付款，一切都在网上管理。

就我个人而言，作为一个极度厌恶清洁的人，我对 Housekeep 达到 6，100%的绝对收入增长并不感到惊讶，这使他们在前 10 名中排名第 6。

FT1000 排名只是他们的最新荣誉。最近，他们被评为 2018 年英国年度小企业；他们也是 2018 年 英国商业技术奖  [中“最令人印象深刻的增长”类别的获奖者。](https://housekeep.com/blog/2018/11/21/housekeep-wins-most-impressive-growth/)

Housekeep 自己也承认，仅仅在他们的邮编范围内，就有 152 家其他的家庭清洁公司——然而，我觉得，有了这样的成就，他们的未来确实非常光明。

##### **管家+ Python**

Housekeep 的开发者被期望在[Python](https://stxnext.com/services/python-development/)或者  [JavaScript](https://stxnext.com/services/javascript-development/) 方面有很强的技能。他们选择的 Python 框架是 Django 也是我们个人的最爱之一。

Housekeep 还用 Python 托管了许多  [GitHub 库，其中一个有 711 个分叉！](https://github.com/HousekeepLtd?utf8=%E2%9C%93&q=&type=&language=python)

##### **Festicket + Python**

#### 6.可信集团

[https://trustly.com/](https://trustly.com/)

![trustly_group.png__730x250_q85_crop_subsampling-2_upscale](img/0ec1253b81f9559903878dd453e4db91.png)

*   排名第 482 位
*   绝对收入增长:362%
*   国家:瑞典
*   板块:金融科技

两家金融科技公司完成了我们的名单，其中第一家是总部位于瑞典的 Trustly。

凭借 362%的收入增长，他们在榜单上排名第 482 位。

他们展示自己的方式，Trustly 的基础是一种渴望——渴望一个更简单的时代，在这个时代，将钱从一个人转移到另一个人是毫不费力的。你只需要一个钱包和一些现金。

现在，在你的银行账户和你真正想付款的人之间似乎不缺少中间人，例如在网上购物时。

这就是信任的用武之地。这是一种支付方式，让你购物和从你的银行账户支付没有任何麻烦。

他们的使命宣言非常中肯:“在 Trustly，我们努力让在线支付变得简单。”

##### **可信群+ Python**

Trustly 的两个用 Python 编写的 GitHub 库都值得注意:

*   第一个是[Trustly-client-python](https://github.com/trustly/trustly-client-python)，“使用 Python 与 Trustly API 通信的示例实现”；

[高级数据库开发人员](https://trustly.com/en/careers/jobs/senior-database-developer-stockholm-8ffbdcce-90a7-46bd-8153-f4d0cd02eec4/) 的候选人，如果他们了解 Python，将大大提高他们的胜算，  [DevOps 工程师](https://trustly.com/en/careers/jobs/devops-engineer-stockholm-743d5da5-ffda-4293-b1c9-186875fe1453/) 和  系统工程师  都必须有用 Python 或类似语言开发的经验。

#### 7\. Zopa

[https://www.zopa.com/](https://www.zopa.com/)

![zopa.png__730x250_q85_crop_subsampling-2_upscale](img/1972402de1a6782330991af874e3afb1.png)

*   **排名第 571 位**
*   **绝对收入增长:304%**
*   **国家:英国**
*   **板块:金融科技**

最后但肯定不是最不重要的，这是一家我们已经在 STX Next 博客上介绍过的公司，在我们的[17 家在其技术堆栈中使用 Python 的金融科技公司中](/stx-new-blog/top-15-fintech-companies-include-python-their-tech-stack/)。

最重要的是，他们和我们同龄！

Zopa 成立于 2005 年，它“建立了有史以来第一家 P2P 贷款公司，为人们提供更简单、更有价值的贷款和投资。”

使用 Zopa 的平台，你既可以作为客户借钱，也可以作为贷款人投资。在每种情况下，资金都是分散的，因此没有一个客户拥有贷款人投资的 1%以上。

自 Zopa 成立以来，超过 76，000 名投资者和机构使用该平台提供了总计超过 39.2 亿美元的贷款。

今年，由于其 304%的增长，金融时报将其列为第 571 位。

在不久的将来，Zopa 计划在  [推出一家基于公平服务的银行](https://blog.zopa.com/2018/12/04/zopas-launching-bank/) ，最终赢得客户的信任——这也是商业街银行近年来一直在失去的信任。

##### **汤+蟒蛇**

在软件开发方面，Zopa 与之前参赛作品的不同之处在于他们对开源的承诺。

用他们自己的话说:

“在 Zopa，我们数据部门喜欢使用开源工具，尤其是 PyData stack，其中包括 jupyter、Pandas 和 scikit-learn 等工具。”

如果你还需要一个理由来解释为什么  [Python 和 fintech](/stx-new-blog/web-app-speed-website-performance/) 完美地结合在一起，这就是了。

Zopa 的开发实力在不断增长。他们正在积极寻求建立自己的 Python 开发人员库，专门寻找重视  [单元测试](/stx-new-blog/measurable-benefits-unit-testing/) 以及集成、验收和端到端测试的程序员。

对测试的关注并不奇怪。毕竟，在金融领域，质量和安全是让企业跻身 FT1000 强的长期成功的关键。
